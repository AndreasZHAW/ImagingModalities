How it Works

Students (phone): Open the student URL or scan the QR code → select modality → move sliders → Submit.

Instructor (projector): Open Beamer tab. Use the 3-way toggle to switch between student data, reference data, and the comparison view with arrows.

Secret autofill: Tap "Group Activity" five times quickly → instructor panel.

The 3-Way Toggle

👥 Student DataShows exactly what each group submitted.
🔬 ReferenceShows the research-accurate values from literature — what the answer "should" look like.
↕ CompareShows both simultaneously. An arrow from the reference position (dashed circle) to the student position reveals discrepancies. Green = correct, orange = differs.
